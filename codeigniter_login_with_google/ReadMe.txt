Installation Instructtion:
==================================================
1. Import the user.sql file into the database of your CodeIgniter application.
2. Move all files to the same directory of your CodeIgniter application
3. Open the "application/controller/User_authentication.php" file and change the $clientId, $clientSecret, and $redirectUrl variable value according to your API Credentials.
4. Open the URL (http://localhost/project_folder_name/user_authentication) on the browser and test the login with Google account.
